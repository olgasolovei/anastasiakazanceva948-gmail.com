Feature: Моніторинг стану здоров’я
  Scenario: Відображення показників здоров’я працівників на дашборді
    Given користувач авторизований у системі
    When сенсор надсилає нові дані про здоров’я
    Then дашборд оновлює інформацію протягом 5 секунд

Feature: Миттєві сповіщення про ризики
  Scenario: Сповіщення надсилається після виявлення ризику
    Given працівник має критичні показники стану здоров’я
    When система виявляє ризик
    Then сповіщення надсилається на мобільний пристрій та e-mail протягом 5 секунд

Feature: Формування звітів
  Scenario: Генерація щомісячного звіту про інциденти
    Given користувач вибирає період звіту та підрозділ
    When натискає кнопку "Згенерувати звіт"
    Then система формує PDF та Excel звіти з графіками та таблицями протягом 10 секунд

Feature: Налаштування IoT
  Scenario: Додавання нового сенсора без переривання роботи
    Given користувач-інженер авторизований у системі
    When додає новий сенсор
    Then сенсор додається та починає надсилати дані без переривання роботи системи

Feature: Безпека даних
  Scenario: Авторизований доступ
    Given користувач вводить логін і пароль
    When користувач натискає "Увійти"
    Then доступ надається тільки після успішної аутентифікації та відповідної ролі
