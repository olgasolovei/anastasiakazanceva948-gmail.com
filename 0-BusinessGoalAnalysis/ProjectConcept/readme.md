# Положення про концепцію проекту

**Для:** кінцеві користувачі — майстри та інженери будівельних майданчиків, а також менеджери з охорони праці.  

**Який:** потребують ефективного інструменту для моніторингу стану здоров'я працівників та попередження аварій на майданчику.  

**Ця:**  HealthManager

**Належить:** категорії веб-додатків для управління будівельними процесами та безпеки праці.  

**Який(а):** забезпечує збір та аналіз даних з IoT-пристроїв, автоматично визначає ризики для працівників, інформує відповідальних осіб про потенційні небезпеки в реальному часі та допомагає приймати своєчасні рішення для запобігання аваріям.  

**На відміну від:** традиційних методів контролю безпеки на будівельних майданчиках, які базуються на ручних перевірках, паперовій документації та запізнілому реагуванні на інциденти.  

**Наш продукт:** пропонує інтегрований цифровий підхід з використанням штучного інтелекту та IoT для своєчасного виявлення ризиків, автоматичного сповіщення та покращення безпеки працівників, що зменшує кількість аварій та підвищує ефективність управління ризиками.  

## Бізнес-вимоги
1. Автоматичний збір даних з IoT-пристроїв на будівельному майданчику.  
2. Аналіз стану здоров'я працівників та прогнозування потенційних ризиків.  
3. Миттєве сповіщення відповідальних осіб про небезпечні ситуації.  
4. Інтуїтивно зрозумілий веб-інтерфейс для перегляду стану майданчика у реальному часі.  
5. Збереження історії даних для подальшого аналізу та звітності.  
6. Висока надійність та доступність системи понад 99% часу.  
7. Використання AI для підвищення точності прогнозування ризиків понад 90%.
